Deck Studio Local

Contenu :
- index.html
- styles.css
- main.js

Fonctions principales :
- création de decks de cartes en local
- sauvegarde automatique dans le localStorage
- écran d'accueil avec decks sauvegardés
- import/export JSON
- export PNG / JPEG / PDF
- recto / verso par carte
- ajout de texte, images, formes, lignes, cadres
- déplacement, redimensionnement, rotation, bordures, couleurs, polices
- duplication / suppression de carte et d'élément
- suppression clavier : Delete / Backspace
- annuler / rétablir
- grille, zone sûre, calques, verrouillage, modèles visuels

Bibliothèques gratuites utilisées via CDN :
- html2canvas
- jsPDF

Important :
- Pour les exports PNG/JPEG/PDF et les Google Fonts, il faut une connexion internet si tu gardes les CDN.
- Si tu veux du 100% hors ligne, il faudra télécharger ces bibliothèques et les héberger localement dans le projet.

Lancement :
- ouvre index.html dans ton navigateur
- idéalement via un petit serveur local pour plus de stabilité
