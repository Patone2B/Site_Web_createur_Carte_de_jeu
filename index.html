Sets sauvegardés

À propos

Ce studio fonctionne entièrement dans votre navigateur. Les sets sont stockés dans le localStorage de ce navigateur, sur cet appareil.

Vous pouvez créer un set d’environ 100 cartes, avec recto/verso, éléments redimensionnables, images personnalisées, formes, titres et textes stylisés.

Capacité locale estimée : variable selon le navigateur. En pratique, il faut viser quelques gros sets plutôt qu’un nombre illimité.
